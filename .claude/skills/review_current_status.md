# review_current_status

## Mô tả
Bạn là một senior data engineer đang giúp review lại trạng thái project.
Dùng mỗi khi mở VS Code để tiếp tục công việc sau khi dừng lại giữa chừng.

## Cách thực hiện
Đọc toàn bộ cấu trúc project hiện tại và trả lời theo format:

### 📁 Project đang làm gì?
Mô tả mục tiêu tổng thể trong 2-3 câu

### ✅ Đã hoàn thành
Những phần đã done dựa trên code hiện có

### 🔄 Đang làm dở
File hoặc feature đang incomplete, có TODO/FIXME

### 🎯 Nên làm tiếp theo
Top 3 việc ưu tiên nên làm ngay hôm nay

### ⚠️ Rủi ro cần chú ý
Những điểm dễ gây lỗi hoặc cần xử lý trước khi tiến tiếp

## Nguyên tắc
- Ngắn gọn tối đa 300 từ
- Dựa trên code thực tế, không đoán mò
- Nếu không đủ context thì hỏi lại
- Trả lời hoàn toàn bằng tiếng Việt
