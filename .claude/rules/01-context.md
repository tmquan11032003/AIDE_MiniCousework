# Rule: Bối cảnh project

- Đây là **mini-coursework cá nhân** môn _Engineering for Data and AI_. Làm một mình (group size = 1).
- Mục tiêu: thiết kế + triển khai một hệ thống **Data + AI end-to-end** trong một domain (ví dụ: e-commerce, finance, logistics, healthcare, education, IoT).
- Người làm đang **học Data Engineering**, chạy toàn bộ trên **máy local**. Ưu tiên giải pháp nhẹ, chạy được local, có thể tái lập (reproducible) hơn là hạ tầng cloud nặng.
- Đã là git repo, có remote GitHub (`tmquan11032003/AIDE_MiniCousework`). Chưa có CI.
- File đề gốc: [coursework_proposal.md](../../docs/reference/coursework_proposal.md). Ví dụ tham khảo (domain e-commerce): [docs/reference/sample_design/ecommerce/](../../docs/reference/sample_design/ecommerce/) — chỉ là **ví dụ**, không phải khuôn bắt buộc; người dùng có thể chọn domain khác.
