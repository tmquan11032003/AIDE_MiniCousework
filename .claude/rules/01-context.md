# Rule: Bối cảnh project

- Đây là **mini-coursework cá nhân** môn *Engineering for Data and AI*. Làm một mình (group size = 1).
- Mục tiêu: thiết kế + triển khai một hệ thống **Data + AI end-to-end** trong một domain (ví dụ: e-commerce, finance, logistics, healthcare, education, IoT).
- Người làm đang **học Data Engineering**, chạy toàn bộ trên **máy local**. Ưu tiên giải pháp nhẹ, chạy được local, có thể tái lập (reproducible) hơn là hạ tầng cloud nặng.
- Đây **chưa phải** git repo. Đừng giả định có remote/CI sẵn trừ khi người dùng tạo.
- File đề gốc: [coursework_proposal.md](../../coursework_proposal.md). Ví dụ tham khảo (domain e-commerce): [sample_design/ecommerce/](../../sample_design/ecommerce/) — chỉ là **ví dụ**, không phải khuôn bắt buộc về nội dung; người dùng có thể chọn domain khác.
